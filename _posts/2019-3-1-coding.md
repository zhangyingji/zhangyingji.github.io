1. 完成一个函数，可以把一个字符串中所有的:到.的子串解析出来到一个数组当中。
例如

输入：getResult('My name is::Jerry.My age is: 12.')

输出：['Jerry', '12']

```
const extractStr = (str) => {
    const ret = str.match(/:([^:\.])*\./g)
    return ret.map((subStr) => subStr.replace(/[:\.]/g, '')) 
}
```
